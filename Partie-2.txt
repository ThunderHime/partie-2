Exercice 1
Créer un dossier website dans le dossier Document
réponse: mkdir website

Exercice 2
Créer un dossier et un sous dossier en une seule ligne de commande dans le
dossier website : Memento/assets
réponse: mkdir -p Memento/assets

Exercice 3
Se déplacer dans le dossier nouvellement créé et faire trois nouveaux dossiers en une seule
ligne de commande : css, img, js
réponse:
	-cd Memento/assets
	-mkdir css img js

Exercice 4
Dans le dossier Memento créer le fichier style.css
réponse: echo "" > style.css

Exercice 5
Dans le dossier Memento créer un fichier index.html et copier/coller ceci dedans :
<!DOCTYPE html>
<html>
 <head>
 <meta charset="utf-8">
 <link rel="stylesheet" href="assets/css/style.css">
 <title>Mon Mémento</title>
 </head>
 <body>
 <h1>La ligne de commande</h1>
 <h2>Gestion de documents</h2>
 <p>Pour créer un dossier je dois utiliser cette commande : </p>
 <pre><code></code></pre>
 </body>
</html>
réponse: 
	-nano index.html
	-copier/coller(shift+Insert)
	-ctrl+x
	-y
	-enter
ou
	-echo "(contenu)" > index.html

Exercice 6
Déplacer le fichier style.css dans le dossier css du dossier assets
p4.pdf
réponse: mv style.css assets/css

Exercice 7
Créer un dossier public à la racine de website
réponse: mkdir public

Exercice 8
Déplacer le dossier assets et tous ses fichiers dans le dossier public
réponse: mv Memento/assets public

Exercice 9
Déplacer le fichier index.html à la racine du projet
réponse: mv Memento/index.html .(.=dossier où je me trouve)

Exercice 10
Supprimer le dossier Memento
réponse: rm -rf Memento (-r= permet de supprimer dossier dans le cas de sous-dossier; -f= forcer la suppression même quand il y a des fichierss)
